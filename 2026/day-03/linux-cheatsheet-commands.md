#######Linux commands Cheatsheet###########

###Process Management Cheatsheet commands ####

  1- ps : current shell process
  2- ps -e : All system process
  3- ps -ef :  Full format (UID,PID,PPID)
  4- ps aux : BSD style, CPU and memory


####Real Time Monitoring Commands#####

  1- top : Real time process view
  2- htop : Enhanced, user-friendly
  3- atop : Advanced performance stats


#####Process Search#########

  1- pgrep : Get PID by process name
  2- pidof : PID of running state
